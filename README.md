# Stock Portfolio Tracker 📈

A simple Python-based stock portfolio management system that allows users to track stock investments, calculate portfolio value, and save investment details to a file.

## 📌 Description

This project helps users manage their stock investments through a command-line interface. Users can add stocks, specify quantities, view their portfolio summary, calculate total investment value, and save portfolio data for future reference.

## ✨ Features

* Add stocks to a portfolio
* Automatic stock value calculation
* Portfolio summary display
* Total investment calculation
* Save portfolio details to a text file
* Input validation for stock names and quantities
* Merge quantities when the same stock is added multiple times

## 🛠️ Technologies Used

* Python 3
* Object-Oriented Programming (OOP)
* File Handling
